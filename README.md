vtile-as-json
=============

Experiment providing MapBox vector tiles as json


## Depends

- Mapnik v2.2.0
- protobuf
- node-mapnik vector-tile branch

## Install

```
brew update
brew install mapnik protobuf
```

Check you have Mapnik v2.2.0:

```
mapnik-config --version
2.2.0
```

Then inside this directory do:

```
npm install
```

## Usage

```
./index.js
```
